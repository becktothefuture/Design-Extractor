# Editorial Filtered Work Index

## Direct Evidence

EPIC's page uses a dark editorial shell, category filters with counts, and large visual cards for work browsing.

## Interpretation

This is useful when the portfolio has many projects but still needs to feel hand-selected.

## Reusable Recipe

Anchor the page with a small section label and rules. Put filters in the same compositional system as the cards. Let the cards do most of the visual work.
