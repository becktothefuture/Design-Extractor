# Legible UI In Playful Product Orbit

## Direct Evidence

Luma keeps the event page readable while wrapping it in a bright summer scene.

## Interpretation

This pattern works when the product's value depends on both utility and social mood.

## Reusable Recipe

Start with a real product screen. Add contextual props around it, not over it. Use the surrounding scene to communicate occasion, audience, and emotion.
