# Category Reframe Media Hero

## Direct Evidence

Zoox uses a plain-language category correction above a large rounded media window showing the product in a street context.

## Interpretation

This is useful when users will compare the product to an older category by default.

## Reusable Recipe

Say what the product is not, then what it is. Show the product in use before listing features.
